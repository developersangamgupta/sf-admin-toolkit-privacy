Privacy Policy for SF Admin Toolkit

This Chrome extension does not collect, store, or share any personal user data.

  • No data is stored externally.
  • No data is shared with third parties.
  • All processing happens locally in the user's browser.
  • Salesforce session information is accessed only to enable functionality and is never transmitted outside the browser.

This extension operates only on Salesforce domains and is intended for administrative productivity purposes.
